== Description ==

- This is a free widget allows you to see locations of all visitors. It's also offers to see top countries , top regions , top visitors and recent visitors by geographical location.


- We show every visitor and keep track if they are a new or returning visitor.

- See how many visits does your website receives hourly, daily, weekly, monthly, and yearly

- Know which country do they come from

- Instant data: We start mapping your visitors as soon as you add the code to your site.

- Easy installation and setup. Just one click.

- Many options to set at backend.

- Reset all data whenever you need.


== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/ directory

2. Activate the plugin

3. You will see Website Stats settings from right side of your wordpress menu.

Configuration Options:

Visitor Message : Put your Website Stats Message in this field.


Set hit counter : Numeric value as per you need.
